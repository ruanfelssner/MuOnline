Launcher Personalizado


Requerimentos Básicos: Conhecimentos com qualquer editor de figuras (até mesmo o paint já está valendo), e criatividade.

Material: Precisaremos da seguinte lista de fotos:

- Uma janela principal para o launcher;
- Uma janela de opções para o launcher;
- Um botão Fechar
- Um botão Fechar Pressionado
- Um botão Conectar
- Um botão Conectar Pressionado
- Um botão Aplicar
- Um botão Aplicar Pressionado
- Um botão Opções
- Um botão Opções Pressionado
- Um botão Update (mesmo que seja uma imagem nula, pois não será utilizado)
- Um botão Update Pressionado


Procedimento: Bom, antes de montar um launcher, creio que vocês já tenham utilizado o Builder para criar launchers básicos. E por isso, na janela principal, irei marcar e preencher os seguintes dados básicos de um launcher.



MU LAUNCHER OPTIONS:

Enable Browser - Exibir página no launcher
Browser URL - Endereço da página a ser exibida

Enable Options - Exibir botão de opções (janela/som...)
Show Server Status - Exibir Status do Servidor (on/off)

Name of Server - Nome de seu servidor
Server Hostname/IP - IP ou DNS para se conectar ao seu servidor
Server Port - Porta de seu Connect Server, normalmente 44405



Agora a parte legal, onde informamos os dados para montar nosso skin do launcher. Acesse a aba SKIN no builder e continue o processo.

SKIN:

Use Custom Skin
- Marcar esta opção para utilizar seu skin alternativo

MAIN WINDOW
- Configurações da Janela Principal de seu Launcher

Color
- Cor de fundo da janela, em meu caso vou deixar padrão. (black)

Font Color
- Cor da fonte utilizada, vou manter padrão também. (black)

Main Window Background Bitmap e Main Window Region Bitmap
- Figura utilizada como moldura de sua página principal do launcher. Nesse caso vou utilizar a muc_bkg.bmp editada para este propósito.

Option Window Background Bitmap e Option Window Region Bitmap
- Figura utilizada como moldura de sua página de opções do launcher. Nesse caso vou utilizar a muc_bkg_opt.bmp editada para este propósito.

A partir de agora, é necessário indicar o posicionamento de botões e forms por sobre o launcher. Para indicar tais posicionamentos, utilizei um editor simples de figuras, e fui acertando o posicionamento dos itens aos poucos, até chegar a valores satisfatórios.
