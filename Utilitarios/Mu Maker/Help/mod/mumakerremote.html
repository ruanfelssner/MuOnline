Config remote access in MuMaker

If your SQL is on another PC and you need access remotely whit MuMaker, you need to configure the MuMaker as follows:



Click in Button ConfigMaker



Select Remote and put IP and PORT of HOST that have the SQL Server, OK.

Them you need Config your SQL Server to allow Remote Access.

Remote SQL 2000 OR Remote SQL 2005


Them you need Config your Router and FireWall to allow Remote Access.

Router / FireWall
